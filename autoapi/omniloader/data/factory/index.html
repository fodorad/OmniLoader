omniloader.data.factory¶

Build datasets and schemas from declarative config entries.

This lets datasets be described in JSON/YAML (adapter + constructor args + schema) so tools like the CLI and OmniConfig can assemble a dataset mix without Python glue. Only the registered data adapters are ever constructed — never models.

A dataset entry looks like:

{
  "adapter": "hdf5",
  "args": {"h5_path": "data/mosei.h5", "subset": "train"},
  "schema": {
    "features": [{"name": "video", "feature_dim": 1024, "time_dim": 300}],
    "targets": [{"name": "sentiment", "placeholder": -5.0}]
  }
}
