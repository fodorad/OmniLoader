omniloader.loader¶

The OmniLoader meta-dataset that unifies several disjoint datasets.

OmniLoader concatenates any number of source datasets, each paired with a DatasetSchema, and presents them as one dataset whose every sample follows the shared UnifiedSchema. Keys a given source lacks are filled with placeholder tensors and all-False masks by the SampleUnifier, so a single batch may freely mix samples from different datasets. A MixingStrategy (obtained via OmniLoader.make_sampler()) controls how the datasets are balanced across an epoch.

class omniloader.loader.OmniLoader(datasets: collections.abc.Sequence[SizedDataset], schemas: collections.abc.Sequence[omniloader.schema.spec.DatasetSchema], schema: omniloader.schema.spec.UnifiedSchema | None = None, pad_features: bool = True, transform: omniloader.transforms.Transform | None = None, training: bool = True, seed: int = 0)[source]¶

Bases: torch.utils.data.Dataset

Unify multiple datasets into a single masked, concatenated dataset.

Parameters:

  • datasets – The source datasets. Each must implement __len__ and return a dict from __getitem__.

  • schemas – One DatasetSchema per dataset, declaring what it provides. Merged into a UnifiedSchema when schema is not given.

  • schema – Optional pre-built unified schema. When None, it is the union of schemas.

  • pad_features – Whether the unifier pads/crops sequence values to their declared time_dim. See SampleUnifier.

  • transform – Optional per-sample transform (normalization/augmentation) applied after unification. See omniloader.transforms.

  • training – Whether augmentations in transform are active. Set False for validation/test loaders.

  • seed – Base seed for per-sample augmentation randomness. Combined with the current epoch (see set_epoch()) and the sample index so that augmentation is reproducible and independent of worker count.

Raises:

ValueError – If datasets is empty or its length differs from schemas.

sequence_lengths(key: str) → list[int][source]¶

Return the native sequence length of key for every sample.

Reads each source sample’s raw value (bypassing unification/transforms) and reports the length of its sequence axis, or 0 when the sample lacks the key. Intended to feed LengthBucketBatchSampler.

Note

This iterates the entire dataset once — O(N) reads. For HDF5 back ends prefer computing it once and caching the result.

Parameters:

key – A sequence feature or target name.

Returns:

A list of per-sample lengths, indexed by global sample index.
